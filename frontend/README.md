# EconoLearn - 经济学基础知识科普平台

一个面向大众的经济学基础知识科普平台，用通俗、结构化、可视化的方式，帮助非经济学专业用户理解核心经济学概念。

## 设计理念

- **学术期刊风格**: 采用精致的极简主义设计，强调排版和留白
- **低门槛**: 避开复杂公式，用生活化语言解释概念
- **可交互**: 动态图表让抽象的经济学关系可视化
- **可解释**: 每个概念都有通俗解释和现实案例

## 技术栈

- **前端框架**: Vue 3 + TypeScript
- **构建工具**: Vite
- **状态管理**: Pinia
- **路由**: Vue Router
- **图表库**: ECharts
- **工具库**: @vueuse/core

## 功能模块

### 1. 首页 (Home)
- 项目介绍和核心价值
- 学习路径预览
- 特色功能展示

### 2. 学习页 (Learn)
- 按分类浏览章节（微观/宏观/金融）
- 学习进度追踪
- 难度标签和状态显示

### 3. 章节详情 (Chapter)
- 通俗版 + 标准版解释
- 现实案例分析
- 相关图表链接
- 收藏和标记完成功能

### 4. 图表探索 (Explore)
- 交互式经济学图表
- 参数调整实时预览
- 供需曲线、价格均衡、通胀等

### 5. 学习中心 (Profile)
- 学习进度统计
- 收藏内容管理
- 全部章节状态

## 设计规范

### 字体
- **衬线体**: Crimson Pro - 用于标题和正文（学术感）
- **无衬线体**: DM Sans - 用于 UI 元素（现代感）

### 配色
- **背景**: 米白色 (#f7f4ef) - 纸质感
- **主色**: 赭石红 (#c45d3a) - 强调色
- **文字**: 深灰 (#1a1814) - 高对比度

### 间距
- 采用 8pt 网格系统
- 基础单位: var(--space-4) = 1rem

## 章节内容
